BILL-012, 015~019, 023 — 결제수단 등록 등 테스트

상태: BLOCKED

사유:
    - 다날 측에서 별도의 테스트 환경(Sandbox)과 테스트용 카드번호를 제공하지 않아
    현재는 실결제가 가능한 계정(CPID, Key) 없이 테스트를 진행할 수 없습니다.
    - 따라서 결제창(PG 팝업)까지는 확인 가능하지만
    그 이후 카드 입력·전체 동의·승인 과정은 실제 결제 정보가 필요해 자동화 테스트로 검증할 수 없습니다.

조치:
    현재는 UI 레벨까지만 검증하고 PG 내부 단계는 테스트 제외로 처리합니다.
